MEDIA MARKT

Media Markt is the Europe's number one consumer electronics retailer with more than 800 stores in 14 countries around the globe. I've worked for this client via Avanade as a UX designer and a Front-end developer. This e-commerce site is now live in Portugal and will soon be available in the other 13 countries where Media Markt has stores.

I started this project with the development of a Proof of Concept to compete on behalf of this client against other companies. After earn the project, the first two months I worked on the design of the user experience. Then I developed the responsive e-commerce Front-End. I also helped the Back-End team to integrate it into the solution, and assisted with the development of the UI of the Xamarin Form App.

E-COMMERCE

I was in charge of the UX design of the e-commerce. The tasks included benchmarking, wireframing and interaction design. In this phase, my experience as a developer was key due to the rapid prototyping of the wireframes in HTML and JavaScript that I did to help the client to see a real flow of interaction between the user and the site.

Next, I developed the complete Front-End site and the whole workflow: home site, landing pages, product pages, catalog, checkout process, etc. After this phase, using my experience as a a .NET Back-End developer I helped the Back-end developers team integrate the Front-End in their Microsoft CMS Sitecore solution.

Project Date

Sept. 2015 - March 2016

COMMERCIAL PLANNING

After finishing the development of the e-commerce I worked on the design and development of a responsive website for the department that manages all the Media Markt commercials for the different media (TV, radio, Internet, etc.) using Microsoft Excel and transfering the files to their retail stores via email.

I made this website to provide a common workplace to put all the commercials and make it accessible for every employee at Media Markt. This website is Back-End agnostic and communicate with the server throught REST services.
