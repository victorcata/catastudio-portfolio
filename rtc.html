E-LEARNING IN REAL-TIME COMMUNICATION

Este es mi proyecto de fin de carrera en el que gane la matricula de honor y fui recomendado para dar un seminario en la universidad explicando mi idea y como llegue a realizarla. La idea se me ocurrio como metodo para aprender ingles a través de internet de una forma colaborativa evitando cualquier barrera tecnologica.

WebRTC IN A COLLABORATIVE ENVIRONMENT

Este proyecto hace uso de Web Sockets para establecer una interfaz que permite a una profesor y un alumno poder realizar distintas actividades para el aprendizaje de un idioma. No se ha aplicado ningun diseño gráfico a esta aplicación, es una prueba de concepto por lo que los pantallazos solo demuestra una comunicación entre dos personas.

Pudiendo ser aplicable para cualquier enseñanza online y no solo la enseñanza de un idioma. La intención es realizar una comunicación oral y escrita a traves de internet sin la necesidad de instalarse ningun software externo para ello (como flash, silverlight, etc). Así mismo el profesor puede configurar la interfaz para mostrar que información se muestra a cada alumno, de que manera y en que orden, siendo la aplicación la encargada de gestionar y guiar la comunicación entre alumnos.
